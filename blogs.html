What are the different ways to select an element in the DOM?

There are several ways to select an element in the DOM:

  1. 1. getElementById() to select an element by its unique ID.
  2. 2. getElementsByClassName() to select elements that share the same class name.
  3. 3. getElementsByTagName() to select elements based on their tag name.
  4. 4. querySelector() allows you to select the first matching element using a CSS selector.
  5. 5. querySelectorAll() selects all matching elements and returns them as a collection.
What is the difference between innerHTML, innerText, and textContent ?

Difference Between innerHTML, innerText, and textContent:

The properties innerHTML, innerText, and textContent are used to access or modify the content inside an element, but they function differently. innerHTML retrieves or sets the HTML inside an element, meaning it includes tags and formatting. innerText extracts only the visible text while ignoring hidden elements and applying CSS styles. textContent, on the other hand, retrieves or sets all the text inside an element, including hidden content, but does not preserve HTML formatting.

What is event delegation in the DOM?

Understanding Event Delegation:

Event delegation is a technique in JavaScript where a parent element is used to handle events for its child elements, even if they are added dynamically. Instead of adding event listeners to multiple individual elements, you attach a single event listener to the parent, which listens for events bubbling up from its children. This approach improves performance and simplifies event handling in dynamic applications.

What is event bubbling in the DOM?

Concept of Event Bubbling:

Event bubbling is a process in the DOM where an event that occurs on a child element propagates upward to its parent elements. When an event is triggered, it first executes on the target element and then moves up the hierarchy, triggering event listeners on parent elements along the way. This behavior allows higher-level elements to capture and respond to events originating from their children. If needed, bubbling can be stopped using methods like event.stopPropagation().

How do you create, add, and remove elements using JavaScript?

Creating, Adding, and Removing Elements in JavaScript:

In JavaScript, elements can be created, added, and removed dynamically. To create an element, you define it in JavaScript before inserting it into the DOM. Once created, the element can be added to a parent element in the document to make it visible on the webpage. If an element is no longer needed, it can be removed from the DOM to free up resources and maintain a clean structure.
